## 技术架构

### 后端服务
```
app/
├── api/
│   ├── auth.py          # 认证API
│   ├── projects.py       # 项目管理API
│   ├── characters.py    # 角色管理API
│   ├── chapers.py       # 章节管理API
│   ├── outline.py       # 大纲管理API
│   ├── style.py         # 风格和AI味API
│   └── schemas.py        # 所有Pydantic schemas
├── models/
│   └── models.py         # SQLAlchemy数据模型
├── services/
│   ├── auth_service.py   # 认证业务逻辑
│   ├── redis_client.py    # Redis客户端
│   ├── vector_db.py     # 向量数据库客户端
│   └── cache_service.py  # 缓存服务
└── database.py           # 数据库连接
└── main.py              # FastAPI应用入口
```

### 前端服务
```
frontend/
├── src/
│   ├── pages/           # 页面组件
│   │   ├── Login.tsx
│   │   ├── Register.tsx
│   │   ├── ProjectList.tsx
│   │   ├── ProjectDetail.tsx
│   │   ├── CharacterManagement.tsx
│   │   └── ConsistencyCheck.tsx
│   ├── components/        # UI组件
│   ├── store/            # 状态管理(Zustand)
│   ├── services/         # API调用
│   └── lib/             # 工具函数
├── public/             # 静态资源
├── package.json         # 依赖配置
└── vite.config.ts       # 构建配置
```

## 项目亮点

1. **完整的AI写作系统**: 从大纲规划到章节生成,全覆盖网文创作流程
2. **智能风格控制**: 确保全书风格一致
3. **AI味降低**: 通过多种策略减少AI生成痕迹
4. **爆款结构参考**: 基于成功网文的结构模板
5. **角色管理**: 可视化角色关系和设定
6. **实时保存**: 防止创作内容丢失
7. **多版本生成**: 为作者提供更多选择
8. **一致性检测**: 自动检测风格和逻辑问题
